Data List

목록은 텍스트와 이미지의 연속적이고 수직적인 데이터를 나타냅니다.
PC와 모바일의 화면 크기 및 브라우저의 크기에 따라 반응형으로 최적화될 수 있도록 노출합니다.

Anatomy

Options

Total value

총 항목 수와 현재 포커스가 있는 항목이 표시됩니다.

Sorting

정렬 방법을 업데이트순으로 선택할 경우 최근에
업데이트된 데이터부터 위에서 아래로 정렬됩니다.
가나다순으로 할 경우 업데이트된 지 오래된
데이터부터 위에서 아래로 정렬됩니다.

Icon

사용자가 로그인을 했을 경우에 나타는 기사 스크랩 관련 아이콘입니다. 외부 공유 아이콘과 사용자 개인이 보관할 수 있는 북마크 아이콘이 있습니다.

Hover

PC 디바이스에서 사용할 때 마우스 오버 시 제목은
밑줄과 회색 컬러로 표시해 줍니다.

Usage

Title length
  • 뉴스 제목은 전체 노출할 수 있도록 최대 2줄에서 3줄까지 노출합니다.
  • 게시물 제목의 경우에는 노출 가능한 범위를 넘어가면 말줄임 표시합니다.
  • 모바일에서는 제목은 최대 3줄로 하고 본문은 노출하지 않습니다.
