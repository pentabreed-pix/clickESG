Box Button

박스 버튼은 상자 형태의 가장 일반적인 버튼입니다.
화면의 콘텐츠 영역이나 하단 영역에 배치되어 화면에서 중요한 작업을 실행하기 위한 버튼으로 사용됩니다.

Anatomy

Options

  • Label and Icon

    버튼에는 항상 레이블이 있어야 합니다.
    아이콘은 필요할 때만 사용하고 레이블 텍스트와
    강력한 연관성이 있을 때만 사용합니다.
    보편적으로 이해되는 아이콘만 사용하는 경우는
    레이블 없이아이콘만 단독으로 사용 할 수 있습니다.

  • Variant

    중요도는 Primary로 갈수록 높아지며,
    Line으로 갈수록 낮아집니다.
    화면 내에서 액션의 중요도에 따라
    버튼의 variant를 다르게 사용합니다.

  • Primary variant

    강한 강조나 중요한 역할을 하는 버튼일 경우에 사용합니다. Primary 버튼은 한 화면에서 너무 많이 사용되지 않도록 제한적으로 사용해야합니다.

    또한, Primary 버튼은 단독으로 하단에 고정하여 사용할 수 있습니다.

  • Secondary variant

    보조 버튼은 강조를 낮추기 위한 것입니다.
    주로 입력폼의 확인/등록의 동작을 하기 위해 사용됩니다. 단독으로는 사용되지 않으며 입력 폼과 같이 그룹으로 사용해야 합니다.

  • Action Variant

    버튼의 상호작용을 목적으로 하는 버튼입니다.
    주로 이전(취소)이나 링크로 사용됩니다.

  • Disabled

    비활성화된 상태로 버튼은 동작이 존재하지만 해당
    상황에서는 사용자가 클릭 할 수 없습니다.

  • Loading

    사용자가 버튼을 눌렀을 때 버튼 내부에서 진행 상태를
    알릴 수 있습니다. 이는 진행이 빠르게 진행되어야 할 때 (5초 이하)에만 사용해야 합니다.

  • Icon

    레이블을 보조하기 위해 아이콘을 사용할 수 있습니다.
    아이콘 두개 동시 사용은 권장하지 않습니다.

  • On/Off states

    이 버튼은 사용자가 탭 할때 마다 YES / NO로 상태를
    변경 할 수 있습니다.

Size

버튼은 소형, 중형, 대형, 초대형의 네 가지 크기로 제공됩니다. 중형 크기는 기본이며 가장 자주 사용되는 옵션입니다.
페이지 내에서 중요도의 계층을 만드는 데 사용해야 합니다. 필요한 경우에 따라 환경에 맞게 버튼 크기는 임의로 변경 가능합니다.

Behaviors

Focus

PC 환경에서 버튼에 Hover 할 경우 커서가
포인터 모양으로 바뀝니다.

Flexible width

PC는 버튼별로 특정 고정 값을 사용합니다.
Tablet과 Mobile은 해상도 내에서 각각
좌우 마진값(24px/16px) 유지해서 가변으로 적용됩니다.

Overflow

Label은 1줄을 초과하지 않도록 작성되어야 합니다.

Usage

Guideline
  • 버튼이 두 개 이상 나열될 때에는 균등한 비율을 권장합니다.
  • 입력폼과 사용할 경우와 Action 버튼이 나열되는 경우는 레이블의 좌우 마진 값으로 버튼의 넓이를 변경할 수 있습니다.

  • 나란히 나열되는 경우 액션 버튼의 우선순위를 고려하여 배치합니다.

  • Primary버튼을 나열할 때에는 균등한 비율로만 배열 가능합니다.

  • 동일한 위계의 서브 액션이 나열되는 경우, Line 사용을 권장합니다.

  • Primary버튼과 Secondary버튼은 나열이 불가능합니다.

High-contrast mode
  • 배경과 최대 대비를 보장하려면 밝은 배경과 이미지에는 정적 검정을 사용하고 어두운 배경과 이미지에는 정적 흰색을 사용합니다.
  • 대비가 많이 다른 복잡한 이미지 위에 정적 구성 요소를 배치하지 않습니다.

테마에 관계없이 Primary버튼 Label은 white를 사용합니다.

다크 테마에서 Primary버튼 Label의 컬러는 반전되지 않습니다.

Fixing the bottom (Only on the app)
  • 앱 화면에서는 항상 화면 하단에 버튼을 고정하고 있습니다.
  • 화면의 버튼 영역에 위치한 이 버튼은 가장 중요한 동작을 실행하는 목적으로 사용됩니다.
  • 버튼과 내용이 겹쳐질 경우를 위해서 그라데이션을 적용합니다.
