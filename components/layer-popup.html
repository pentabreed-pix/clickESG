Layer Popup

팝업은 사용자에게 중요한 정보를 보여주거나 사용자 선택을 요청하는 구성 요소입니다.
창이 띄워진 경우, 이 창을 종료하지 않는 이상 원래 창 화면을 작동할 수 없습니다.

Anatomy

Element patterns and Types

Value Usage
Default 기본 알림 팝업으로 사용자 확인이 중요할 때 사용됩니다.
Bottom Sheet 선택 또는 설정에 있어 화면 이동이 발생하지 않도록 하단에 고정된 시트로,
내용 변경이 거의 없어 내용에 맞게 크기가 고정되어 있는 기본형 시트입니다.
Layer Pop 레이어 팝업은 주로 사용자가 화면에 진입하자마자 세부 정보를 노출하여 관심을 집중할 수 있도록 하는것이 목적입니다.
Full Pop 큰 사이즈의 레이어 팝업으로 내용이 많을 경우 사용하며 정해진 길이 초과시 스크롤이 생성됩니다.

Usage

Header

하단 버튼에 따라서 상단 닫기 아이콘 버튼을 노출하지 않을 수 있습니다.

Resizing

내용이 정해진 시트 높이를 초과하면 창 높이가 올라가면서 내용을 스크롤 할 수 있습니다.

Scroll area

콘텐츠가 최대 높이나 설정된 높이보다 길면 상단 영역이 자동으로 스크롤 영역이 됩니다.

Landscape
  • 팝업 가로 모드로 변환할 때 스크롤이 생성됩니다.
  • 바텀 시트에서 하단 고정 버튼을 사용하는 경우 모드 전환이나 스크롤에 관계없이 플로팅 버튼의 위치가 고정되어야 합니다.
