진행률 표시기는 사용자에게 업로드 또는 다운로드의 진행 상황을 실시간으로 보여주고 100%가 되면 자동으로 없어집니다.
지표와 함께 사용되는 컨텍스트 및 구성 요소에 따라 적절한 유형을 사용합니다.

Value Usage
Circular Type 기본적인 로딩로 원형의 반복 애니메이션 단일 형태이며 각 사용처에 맞게 로딩 사이즈를 사용합니다.
Linear Type 웹뷰에서 사용되며, 화면 상단에서 얇은 바 형태의 로딩 애니메이션입니다.

Usage

Circular type

  • 전체 데이터 로딩이 필요한 새로고침 또는 페이지 이동 할 경우에는
    브라우저 센터에 노출됩니다.

  • 게시물을 등록 중인 경우 버튼에서 노출됩니다.

Linear type

  • 웹 뷰에서 노출될 때에는 브라우저 상단 GNB 위에 노출됩니다.

  • 진행 상태를 나타낼 때에도 사용합니다.
