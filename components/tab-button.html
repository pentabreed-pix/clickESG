Tab Button

콘텐츠 영역 상단에 위치하여 여러 섹션을 쉽게 전환시키는 컴포넌트 입니다.
탭의 사이즈는 상황에 따라 변경 사용할 수 있으며 탭 선택 시 화면 내의 특정 영역의 콘텐츠가 변경됩니다.
탭 타입은 공통 탭과, 예외 케이스 탭으로 나누어 사용합니다.

Anatomy

Options

Label

탭 항목에는 접근성을 위한 레이블이 있어야 합니다.

Stage

페이지 내에서 중요도의 계층을 만드는 데 사용해야 합니다.
Depth의 숫자가 작을수록 중요도가 높아집니다.

States

선택된 상태의 탭 항목은 현재 탭 항목을 보여줍니다.
한 번에 탭 항목 하나만 선택할 수 있습니다.

Pading

탭 항목 컨테이너로 탭의 영역을 나타냅니다.
사용자가 클릭 가능한 범위로 설정 되어야 합니다.

Margin

레이블과 레이블 간의 공간을 조정할 수 있습니다.
숫자는 텍스트 사이의 공간 넓이를 나타냅니다.

Element Type

항목 수

제공할 탭 항목의 수를 선택할 수 있습니다. 각 해상도 컨텐츠 영역을 넘어갈 경우 가로스크롤을 사용합니다.

측면 여백

기본적으로 탭 항목은 컨테이너의 시작(왼쪽)에 정렬됩니다. 카테고리에 따라 여백 유무가 달라질 수 있습니다.

Usage

표시될 콘텐츠가 없는 경우

탭을 삭제하거나 탭을 선택했을 때 패널에 콘텐츠가 없는 이유에 대한 설명을 제공하여 사용자가 혼동을 느끼지 않도록 해야 합니다.

여러 개의 탭을 중첩하는 경우

불가피하게 탭을 중첩해야 하는 경우 중첩된 탭의 스타일이 시각적으로 분명하게 구분되어야 합니다.

탭의 항목 수가 많을 경우

PC에서 모든 탭이 영역의 너비에서 넘쳐서 표시될 수 없는 경우에는 숨겨진 탭을 탐색할 수 있는 좌/우로 이동할 수 있는 컨트롤 요소를 제공해야 합니다.
