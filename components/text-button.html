Text Button

링크는 다른 서비스/애플리케이션, 한 서비스 내의 다른 화면, 한 화면 내의 다른 섹션 등으로 이동하는 데 사용됩니다.
중요도에 따라 컬러와 사이즈(높이값)를 다르게 사용하고 선택적으로 아이콘과 결합해 사용합니다.

Anatomy

Options

Text only

레이블만 사용하는 버튼으로 너비는 버튼 레이블에
따라 결정되며, 필요에 따라 사이즈를 사용합니다.

Text + Icon

레이블과 아이콘을 결합해 사용하는 버튼으로,
아이콘은 레이블의 좌측과 우측에 모두 위치할 수 있습니다.

Underlined

텍스트(레이블)에 밑줄을 사용하는 버튼으로,
강조돼야 할 버튼에 사용합니다.

Usage guidelines

화면 판독기 지원

배치와 언어를 주의 깊게 살펴보고, 화면 판독기를 사용하여 탐색하는 사용자를 고려합니다.

맥락을 전달하고 링크가 어디로 갈지에 대해 명확하게 합니다.

화면 판독기를 사용하는 사람들은 "자세히 알아보기" 또는 "여기를 클릭하세요"와 같은 매우 일반적인 링크 문구는 전달하지 못합니다.

문장 내 사용

문장 내에서 사용할 때에는 버튼은 마치 문단의 일부인 것처럼 작성될 수 있습니다.
랜딩 경험의 가치와 기대를 전달하기 위해 레이블 맥락을 끊어주어야 합니다.

레이블은 연결된 목적지 정보를 예측할 수 있는 내용으로 제공합니다.

텍스트가 줄바꿈될 정도로 길어지는 것은 피해야 합니다.

직관적인 경험

해당 링크가 어디로 갈지 또는 무엇을 할지 정확히 식별할 수 있도록 명확하게 표현합니다.

사용자가 가장 유용한 다음 단계를 빠르게 찾을 수 있도록 합니다.

여러 문장이나 문단의 끝에 "자세히 알아보기"라는 일반적인 독립형 링크를 연속해서 사용하는 것은 피하는게 좋습니다.
