Content design

UX 작성 및 정보 설계를 통해 UI를 사용자가 필요할 때 필요한 정보를 쉽게 사용할 수 있도록 디자인 합니다.

콘텐츠 디자인이란?

1. Alt text

효과적인 대체 텍스트는 화면 판독기 사용자를 위해 이미지의 의미와 맥락을 설명합니다.

2. Global writing

글로벌 쓰기는 여러 문화에 대해 포괄적으로 쓰고 현지화를 위해 최적화합니다. 읽고, 이해하고, 번역하기가 더 쉽습니다.

3. Style guide

UI 텍스트는 누구나, 어디서나 이해할 수 있어야 합니다.

모범 사례

대체 텍스트는 짧게

대체 텍스트의 권장 길이는 140자입니다.
대체 텍스트가 너무 길면 일부 화면 판독기에서 잘려
나갈 수 있으며 사용자 경험에 좋지 않습니다.

차트와 그래프는 요약해서 작성

대체 텍스트는 차트와 그래프에 특히 중요합니다.
일반적으로 모든 데이터 포인트를 자세히 설명하는 대신
데이터를 요약하여 대체 텍스트를 작성합니다.

그래프를 생성한 데이터에 대한 링크가 있으면 제공합니다.

평범한 언어를 사용

제목을 사용하여 콘텐츠를 구성하고, 문구를 일대일로 대화하는 것처럼 대화적인 어조로 작성합니다.

전 세계적으로 사용할 수 있도록 명확한 문구로 작성합니다.

사용자 시각의 흐름

제목과 부제목을 사용하여 정보의 우선순위를
지정하고 그룹화합니다.

사용자가 액션을 훑어보거나 서두를 때 이러한 구성은
실수와 의도치 않은 액션을 피하는 데 도움이 됩니다.
