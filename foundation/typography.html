Typography

웹 인증 글꼴 패밀리, 글꼴 두께, 문자 간격을 사용하여 일관된 디자인을 만들 수 있도록 합니다.
타이포그래피 시스템은 제목과 텍스트 유형을 제공하고 사용법을 지정합니다.

Typeface

연합뉴스 웹/앱에서는 Pretendard을 표준 폰트로 사용합니다.

프리텐다드 서체 정보 : https://cactus.tistory.com/306

Components

Heading

탑뉴스 타이틀에 주로 사용합니다.

Title

뉴스 타이틀에 주로 사용합니다.

Body

뉴스 본문에 주로 사용합니다.

Detail

추가 정보 또는 작은 항목 텍스트에 사용합니다.

Label

구성 요소 내부의 텍스트로 사용합니다.
(예: 버튼, 키워드 등)

Link

문장 내의 몇 단어로 이루어진 텍스트 혹은 메뉴와 같이 독립적인 링크에 사용합니다.

Font weight

모든 페이지 내에서 텍스트의 두께는 Regular, Medium, Semibold, Bold 4가지를 원칙으로 사용합니다.

Line height

Heading and Title

탑뉴스 헤드라인의 경우 1.3배(130%) 행간을 사용합니다. 그 외에는 1.5배(150%) 행간을 사용합니다. 이는 모든 글꼴과 대체 글꼴에 적용됩니다.

Body and Detail

뉴스 본문 태블릿, 모바일에서는 기본 행간 1.5배수가 아닌 예외 케이스로 1.7배(170%)로 행간을 사용합니다.
이는 모든 글꼴과 대체 글꼴에 적용됩니다.

Component

가독성을 보장하기 위해 1.5배 행간를 사용합니다.
이는 모든 글꼴과 대체 글꼴에 적용됩니다.
