Object Styles

Object Style은 세 가지 시각적 요소인 Radius, Border Width, Shadow를 분류하여 전체 UI에서 시각적 일관성을 유지합니다.
버튼 및 텍스트 입력 상자와 같은 디자인 구성 요소 컨테이너에 적용되는 스타일 속성을 정의합니다.

Radius

Small rounding

대부분의 구성 요소에서 가장 자주 사용되는 값입니다.

Default rounding

일반적으로 컨텐츠 카드 및 팝업 메세지와 같이
큰 형태의 박스 구성 요소에 사용됩니다.

ETC rounding

강조가 필요한 부분에 예외적으로 사용하는
라운드 유형입니다.

Full rounding

UI 요소에 사용하는 라운드 값으로 강조되는
구성 요소에 사용합니다.

Border width

Border width
Value Usage
1 px border 가장 일반적인 테두리 너비입니다.
일반적인 테두리(예: 텍스트 필드, 태그, 팝오버) 또는 구분선(예: 테이블, 작은 구분선)으로 사용됩니다.
2 px border 요소의 강조가 필요할 때 이 값을 사용합니다.
3 px border 주로 라인으로 구성된 구성 요소(예: 슬라이더 또는 탭)로 사용됩니다.
